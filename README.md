# 🕵️ Data Detective: Business Mystery Challenge

An interactive data analytics game where users solve real-world business mysteries using data instead of just viewing dashboards.

---

## 🎯 Project Overview

This project simulates a real-world business environment where a retail company faces:

- 📉 Sudden sales drop
- 🚨 Fraud transactions
- 📦 Inventory mismatch
- 🧾 Customer behavior issues

As a **Data Detective Analyst**, the user investigates datasets, finds patterns, and solves business problems using data analytics.

---

## 🎮 Key Features

- 🕵️ Interactive detective-style game
- 📊 Business intelligence dashboard
- 📈 KPIs (Revenue, Profit, Orders)
- 🔓 Level-based case unlocking system
- 🚨 Fraud detection simulation
- 📦 Inventory analysis
- 🏆 Score + Rank system
- 🎨 Dark detective-themed UI

## 🚀 How to Run

### 1. Install dependencies
pip install streamlit pandas plotly

### 2. Run the app
streamlit run streamlit_app/app.py

---

## 🧠 Game Flow

- Start in Home page (story mode)
- Go to Game mode
- Solve detective questions
- Earn score + unlock levels:
  - Level 1 → Sales Analysis
  - Level 2 → Fraud Detection
  - Level 3 → Inventory Investigation
- Unlock Cases page based on level
